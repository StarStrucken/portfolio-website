Web Design

Making websites involves the process of creating and developing a website from scratch or using website building tools. This includes designing the website layout, creating and editing website content, coding the website using languages such as HTML, CSS, and JavaScript, and publishing the website to a web server.

Game Design

Designing a game involves creating and developing the concept and mechanics of a game and determining the target audience. This includes implementing the rules and mechanics, designing levels, characters, storylines, and creating the game's art and sound assets. The goal of game design is to create a fun, engaging experience for players.

Javascript Calculator

Within this Grade 10 project is a calculator that you can access anytime that you'd like! It includes features such as working input of buttons/user input, calculations of several mathematical equations, and order of operations. Have fun calculating away your worries!

Bushido Legacy

This final Grade 11 project was inspired by the Street Fighter game. The intended functionality of the game is to have the user fight an enemy for a period of time using keys to move left and right, attack, jump, and crouch. I am still implementing the enemy AI, so its current state is a 2 player samurai game. The style of the game is Meiji-period inspired, with music, pixel art characters, animations, and backgrounds.

The World's Simplest Game

In this mid Grade 11 assignment, I was supposed to create a multilevel game inspired by the World's Hardest Game, which was trending several years back. The objective is to get to the other side of the level using the WASD keys as a square, and not get hit by the blue or red obstacles. Good luck!
